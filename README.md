### WARBot / WARMusic

Discord の Bot です。       
WARBot には便利機能などがあり、WARMusic は音楽 Bot です。

通常、音楽 Bot は有料でないと音量調節ができませんが、WARMusic は無料ですべてのコマンドが使用できます。

🤖WARBot: https://discord.com/api/oauth2/authorize?client_id=473006066116984842&permissions=8&scope=bot
